# Proyecto Anime API - EA2

Script en Python que consume la API REST de Jikan para obtener información de tus animes favoritos.

## Requisitos
- Python 3.9+
- Librerías: `requests`, `python-dotenv`

## Cómo ejecutar
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar el script: `python main.py`

## Flujo del Proyecto
Este proyecto cumple con la arquitectura REST, manejo de excepciones y uso de contenedores Docker para su despliegue automatizado.
